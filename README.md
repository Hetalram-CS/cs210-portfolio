# CS 210 Portfolio Project – Grocery Tracking Program

## Project Overview  
In this project, I created a Grocery Tracking Program using C++. This program reads grocery item data from a file and counts how many times each item appears. It also allows the user to search for an item and shows the frequency. This helps to understand shopping patterns and manage grocery lists better.

## What I Did Well  
In this project, I wrote simple, clear code. I was able to use file I/O correctly, and my program runs without errors. I also made the program easy for the user to understand.

## Areas for Improvement  
I think I can improve my code by making it more efficient and adding better input validation. I can also add more features like sorting the data or improving the display. These changes will make my program stronger and more professional.

## Challenges  
The most difficult part for me was working with file handling in C++. It was confusing at first, but I practiced using zyBooks examples and tested my code many times. That helped me understand it better.

## Transferable Skills  
From this project, I learned C++ programming, file handling, debugging, and problem-solving. These skills will help me in future courses and real-world projects.

## Maintainability  
I tried to keep my code simple and readable by using clear variable names and organizing the code properly. This makes it easier to understand and update later.
